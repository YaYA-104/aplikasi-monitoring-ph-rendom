Dashboard Monitoring Kondisi Tanah Secara Real-Time

Aplikasi web ini menampilkan data hasil pengujian pH tanah, suhu tanah, dan kelembaban tanah dari prototype monitoring berbasis Internet of Things. Data digunakan untuk membantu pemantauan lahan pertanian secara lebih mudah, terukur, dan terdokumentasi.
